# Binil Sanil Liby — Portfolio

> Personal portfolio website built with React. Features a dark, elegant design with smooth scroll navigation, animated sections, and a responsive layout.

## Live Preview

Run locally at `http://localhost:3000`

## Tech Stack

- **React** (Create React App)
- **CSS-in-JS** (inline styles)
- **Google Fonts** — Playfair Display + DM Sans
- **IntersectionObserver API** — scroll-triggered fade-in animations

## Project Structure

```
my-portfolio/
├── public/
│   ├── index.html
│   └── binil.jpg          # Profile photo
├── src/
│   ├── Portfolio.jsx      # Main portfolio component
│   ├── index.js           # React entry point
│   └── index.css
├── package.json
└── .gitignore
```

## Sections

- **About / Hero** — Intro, photo, stats
- **Experience** — Work history timeline
- **Projects** — Featured ML/AI projects with GitHub links
- **Skills** — Technical skills by category
- **Contact** — Email, phone, LinkedIn, GitHub

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start
```
